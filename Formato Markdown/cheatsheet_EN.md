## ðŸ”– **Headings**

Use `#` for headings. Increase the number of `#` for subheadings.

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### **Rendered Example:**

# Heading 1  
## Heading 2  
### Heading 3

## ðŸ“ **Text Formatting**

Format text using asterisks (`*`) or underscores (`_`).

```markdown
**Bold Text**
*Italic Text*
***Bold and Italic***
~~Strikethrough~~
```

### **Rendered Example:**

**Bold Text**  
*Italic Text*  
***Bold and Italic***  
~~Strikethrough~~

## ðŸ“‹ **Lists**

```markdown

### **Rendered Example:**

## ðŸ”— **Links**

Create links using the following syntax:

```markdown
[Link Text](https://example.com)
```

You can also define reference links:

```markdown
This is a [reference link][1].

[1]: https://example.com
```

### **Rendered Example:**

[Link Text](https://example.com)  
This is a [reference link][1].

[1]: https://example.com

## ðŸ› ï¸ **Code Blocks**

### **Inline Code**

Use backticks for inline code:

```markdown
The command `docker run` starts a container.
```

### **Code Blocks**

Use triple backticks for multi-line code blocks. Specify the language for syntax highlighting.

```markdown
```python
# Python example
print("Hello, world!")
```
```

### **Rendered Example:**

```python
# Python example
print("Hello, world!")
```
